# invent<sup>*ory*</sup>

An app to keep track of all your PCB components

## Planed features:

- import PCB BOM to see what has to be reordered if the user wants to make n PCBs
- import BOM to automatically subtract the parts for n PCBs from the inventory stock
- button to see the datasheet
- more data abt each part (LCSC api delivers more than used)
- edit boxes
- search component by any of its attributes
- better qr code scanning
- view to see how an entire box should be optimally ordered to physically find stuff in it
- view to look up where to put back a package taken out of a box
- display image of the component (asset from lcsc api)
- translate datasheets
- settings to set the backend server without rebuilding
